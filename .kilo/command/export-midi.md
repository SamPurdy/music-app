Export song/chord data to MIDI:

1. Convert chord progressions to MIDI notes
2. Generate a .mid file compatible with Ableton
3. Support multiple tracks (chords, melody, bass)
4. Preserve tempo and key information
5. Include Roman numeral labels as MIDI meta-events

Usage: /export-midi [song-data] [output-path]

The exported MIDI can be dragged directly into Ableton Live as a MIDI clip.
