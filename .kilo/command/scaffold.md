Scaffold a new music app project:

1. Create the project structure from AGENTS.md
2. Initialize React + TypeScript (or Electron if requested)
3. Install core dependencies: tonal, @tonejs/midi, midi-js, ableton-link
4. Set up the music theory engine skeleton in src/lib/music-theory/
5. Create base types in src/types/
6. Add initial tests for music theory calculations

Ask the user about their preferred stack (React web vs Electron desktop) before proceeding.
